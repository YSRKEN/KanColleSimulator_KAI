KanColleSimulator_kai

概要

KanColleSimulator_kaiは艦これの戦闘をシミュレートします。

ヘルプ

  • CUI
  • GUI

対応コンパイラ

CUI(C++)

  • Visual Studio 2015 Update 2
  • Visual Studio 2015 Clang with Microsoft CodeGen(March 2016) (clang 3.7)

mingw-gccやmingw-clangはコンパイルはできると思いますが、文字コードまわりの影響で上手く動かないと思います。

GUI(C#)

  • Visual Studio 2015 Update 2

WindowsFormを使用しています。
